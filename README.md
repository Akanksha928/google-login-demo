# Google Login Demo

A cross-platform Google OAuth 2.0 login demo built with React + Vite.

## Tech Stack

| Layer | Tech | Purpose |
|------|------|--------|
| UI | React + Vite | Fast frontend development |
| Auth | @react-oauth/google | Google OAuth (popup-based login) |
| API Calls | Fetch API | Retrieve user profile from Google |
| Styling | CSS | Lightweight styling |

## 🚀 Run Locally

```bash
npm install
npm run dev
